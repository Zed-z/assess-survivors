# Assess Survivors
Survive hordes of enemies and make choices along the way!

----

# Survive!
Survive against an ever-increasing horde of enemies! Each one has different skills and movement patterns, so watch out!

# Choose!
Choose your rewards at each level up! You can choose a sure reward or risk it for a bigger boost! The game will track your selections and help you with choices later on!

# Compare!
Share your results with people online and aim for the leaderboard! Can you make it to the top?

# Controls:
- Move with WASD / Arrow keys
- Shoot projectiles with LMB
- Melee attack with RMB
